EasyContract

Javascript HTML/CSS Bootstrap Node.js Express MongoDB

Source Code

Idea

The vision for the application originated from StartRight's need to streamline and expedite the contract creation process. Following the initial conversation and understanding of their requirements, a collaborative effort was initiated to develop a concept aimed at simplifying the work processes of the student organization. The objective was to create not only a contract generator but an entire application with additional features to enhance the efficiency of StartRight's operations.

Concept

Prior to delving into the concept, extensive research was conducted on existing tools with similar functionalities. Through multiple meetings with the students from StartRight, exploration of additional features and ideas that could serve as meaningful extensions to the application took place. These discussions and considerations shaped the goals for the project. Regular communication and collaboration with StartRight ensured that the application met their exact requirements and expectations.

Process

1. Mockup:


All ideas and considerations from the concept phase were formalized into a structured plan, providing a clear roadmap for development. The meticulous planning phase, including mockups and prototypes, laid a strong foundation for the development process, reducing the likelihood of unexpected challenges.


2. Prototype:


With the concept solidified, the transition to the implementation phase began, creating the first prototype. This crucial step allowed for the visualization of the application's flow and functionality. During this phase, the decision was made to build the project entirely from scratch, utilizing a carefully selected tech stack.


3. Tech Stack:


The entire project was built from the scratch both backend and frontend, using the Model-View-Controller (MVC) architectural pattern and employing the following tech stack:


HTML, JavaScript, CSS, Bootstrap for front-end development.
Node.js as the open-source server environment, providing a script structure for websites.
Express, an extension of Node.js, serving as a lightweight web framework to handle all requests.
MongoDB, a document-oriented NoSQL database management system, for efficient data organization.
Mongoose, a schema-based solution, simplifying data management.

The use of Node.js and Express allowed for a robust and scalable platform, while MongoDB and Mongoose facilitated efficient data handling.
